# Proyecto Sistema de Autenticación con JWT y Cookies

Este proyecto consiste en el desarrollo de un sistema de autenticación utilizando Express.js, JWT, cookies y middleware para proteger rutas privadas. El objetivo fue crear un servidor capaz de validar credenciales, generar un token seguro, almacenarlo en cookies y permitir el cierre de sesión.

## El Equipo

Benjamín: Encargado de instalar Node.js y las dependencias del proyecto, configurar Express.js, crear server.js e implementar la ruta de login.

Sebastian: Se encargó del diseño de la interfaz, creación de index.html, style.css, script.js y de las pruebas visuales del sistema.

Vicente: Encargado de crear el middleware authMiddleware.js, proteger la ruta privada, implementar el cierre de sesión y elaborar el informe final.

## Archivos del proyecto

* server.js: Archivo principal del servidor Express.js.
* users.js: Arreglo con usuarios ficticios para validar credenciales.
* authMiddleware.js: Middleware encargado de proteger rutas privadas y validar el token JWT.
* index.html: Interfaz principal del sistema.
* style.css: Diseño visual de la página.
* script.js: Lógica del cliente para login, perfil y logout.
* Informe_Sistema_Autenticacion_JWT_Grupo15.docx: Informe completo del desarrollo de la actividad.
